# ABC
ABC (Abstract Basic Class)

## Notations
- @abc.abstractmethod: child class must implement the function/property, otherwise raise error
- @classmethod: can access to class attributes
- @staticmethod: cannot access to class attrubutes
- @property: can access using property-like method (A.property) 
- @<property>.setter: setter func for property, called using (A.property = value) 
- @<property>.getter: getter func for property, called using (A.property)
