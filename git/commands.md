- revert commit  
`git reset --hard HEAD~`
- revert commit and keep changes  
`git reset --soft HEAD~`
- revert commit from github  
`git push origin HEAD --force`
- force push  
`git push --force`
- edit commit message  
`git commit --amend`
- look up commit by hash
`git checkout <branch>`
`git log -p -1 <hash>`
- revert commit
`git revert <hash>`
`git reset --soft <hash>`
